# Smart Parking System

A Smart Parking Management System developed using C programming.

## Features
- Vehicle parking management
- Slot allocation
- Bill generation
- Ticket system
- File handling

## Technologies Used
- C Programming
- Structures
- File Handling

## Project Description
This project helps manage parking slots efficiently using a simple command-line interface. Users can park vehicles, remove vehicles, check parking availability, and generate parking bills.

## Future Scope
- GUI implementation
- Online parking booking
- Database integration
- Mobile application support
